# GitHub Copilot Instructions

## Obecné pokyny pro programování

### Styl kódu
- Používej konzistentní formátování
- Přidávej komentáře k složitým částem
- Používaj smysluplné názvy proměnných a funkcí

### Bezpečnost
- Nikdy nevkládej API klíče nebo hesla do kódu
- Používaj environment variables pro citlivé údaje
- Validuj všechny uživatelské vstupy

### Testování
- Piš testy pro nové funkce
- Používaj deskriptivní názvy testů
- Pokrývej edge cases

### Dokumentace
- Dokumentuj veřejné API
- Vysvětluj složité algoritmy
- Udržuj dokumentaci aktuální

## Specifické pokyny pro technologie

### Clojure
- Preferuj immutable datové struktury
- Používaj threading macros pro lepší čitelnost
- Následuj konvence pojmenování (kebab-case)

### JavaScript/TypeScript
- Používaj ES6+ syntaxi
- Preferuj const/let před var
- Typuj proměnné v TypeScriptu

### Python
- Následuj PEP 8
- Používaj type hints
- Používaj f-strings pro formátování

## Pracovní postupy

### Git
- Piš smysluplné commit zprávy
- Používaj feature branches
- Rebasuj před mergováním

### Code Review
- Kontroluj logiku a čitelnost
- Hledej potenciální bugy
- Navrhuj zlepšení
